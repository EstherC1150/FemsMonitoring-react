# FEMS Monitoring Dashboard

React based FEMS (Factory Energy Management System) monitoring dashboard built with Vite.

## Tech Stack
- React
- TypeScript
- Vite
- Tailwind CSS

## Getting Started

### Prerequisites
- Node.js (v18 or higher recommended)
- npm

### Installation
```bash
npm install
```

### Development
```bash
npm run dev
```

### Build for Production
```bash
npm run build
```
